# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please open a [GitHub Issue](https://github.com/gjczone/pi-shazam/issues/new) with the label `security`.

We will acknowledge receipt within 48 hours and provide an estimated timeline for a fix.

## Responsible Disclosure

We ask that you:

- Do NOT disclose the vulnerability publicly until we have had a chance to address it.
- Provide sufficient detail to reproduce the issue.
- Allow us reasonable time to fix the issue before disclosure.

## Supported Versions

Only the latest release receives security patches.
